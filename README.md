# Color Picker

A simple web application to browse and copy color codes to clipboard.

## Features

- Browse a collection of predefined colors
- Click any color to copy its hex code to clipboard
- Visual feedback with modal notifications and toast messages
- Keyboard accessible (Enter/Space to copy)
- Smart contrast for text readability

## Usage

1. Open `index.html` in a web browser
2. Click on any color card to copy its hex code
3. The copied color will be confirmed in a modal notification

## Tech Stack

- Vanilla JavaScript (ES6 modules)
- HTML5
- CSS3

## Files

- `index.html` - Main HTML structure
- `index.js` - Application logic
- `color-data.js` - Color data array
- `style.css` - Styling
